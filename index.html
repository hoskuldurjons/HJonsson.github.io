Tableau Projects

AirBnB Seattle Area

In this Tableau visualization project, we examine a dataset of over 10 million datapoints from AirBnB listings in the Seattle, WA. area in 2016. The large dataset allows us to explore various details related to zip codes, pricing, and revenue.

The project provides visual information on the number of bedrooms per listing, total revenue by bedroom count, average price per bedroom, zip code prices, and reported revenue for 2016. One of the visualizations presented is a map of the Seattle area that shows the distribution of AirBnB listings by zip codes.

Some interesting findings include:

  • There are more than twice as many single bedroom listings as all multiple bedroom listings combined.
  • Single bedroom listings generated over $63m in revenue, more than all two to six bedroom listings combined ($60.3m).
  • Reported revenue in December 2016 was more than double the revenue reported in January 2016.

Visit my GitHub repository to access the .twbx file or a PDF showcasing the visualization.

Global Covid Vaccine Tracker

The Global Vaccine Tracker Tableau visualization is an interactive tool that displays COVID-19 vaccination progress worldwide. The visualization draws data from the Our World in Data COVID-19 Vaccination Dataset and allows users to select a start and end date from January 1st, 2020 to April 8th 2023, as well as filter by continent and/or country, and answers the question "What share of the population has been partly or fully vaccinated?"

The visualization includes a horizontal bar chart and world map that shows the share of people fully and partly vaccinated for each country. It also features a scatter plot that displays the number of people vaccinated per hundred against each country's GDP per capita

Check out the full visualization on my GitHub repository to access the .twbx file or a PDF showcasing the visualization.

Power BI Project

Data Professionals Survey

In this Power BI visualization project, we analyze the results of a Data Professionals Survey with 504 full-time employees in data-related roles. We investigate various aspects such as respondents' country of residence, job titles, programming languages, average age, work/life balance, and salary.

Some interesting findings include:

  • The treemap shows the distribution of respondents by country of residence. The job titles of "Data Scientist" and "Data Engineer" earn the highest salary and Python is the most popular programming language.
  • On a scale of 1-10, respondents' "Happiness with work/life balance" scored 5.72, while "Happiness with salary" scored 4.26. It is important to look at work/life balance because it is an indicator of job satisfaction and employee well-being.
  • When asked "How difficult is it to break into data?" as a profession, 24.35% answered "Easy", while 23.83 responded "Difficult".

Visit my GitHub repository to access the .pbix file or a PDF showcasing the visualization.
